# Инструкция по установке и запуску проекта

1. Выполнить команду:
```bash
git clone https://github.com/ianmoore228/saasoft-test.git
cd saasoft-test
```
2. Установить зависимости:
```bash
npm install
```
3. Запустить проект локально:
```bash
npm run dev
```
